# PhoneZone - Mobile Shop App

PhoneZone is a mobile shopping application developed using Kotlin and Jetpack Compose. The app allows users to browse smartphones, check product details, select storage and color options, add products to the cart, and place orders.

This project was developed for Mobile Computing practice using modern Android development tools and architecture.

## Features

* Browse available smartphones
* Search and filter products
* View phone details and specifications
* Select storage and color variants
* Add and remove items from the cart
* Update item quantity
* View total cart price
* Order confirmation screen

## Technologies Used

* Kotlin
* Jetpack Compose
* IntelliJ IDEA
* Navigation Compose
* Coil
* ViewModel

## Project Structure

* `MainActivity.kt` - Application entry point
* `Product.kt` - Product model class
* `CartItem.kt` - Cart item model
* `ProductData.kt` - Product data source
* `ShopViewModel.kt` - State and cart management
* `NavGraph.kt` - Screen navigation
* `HomeScreen.kt` - Product listing screen
* `ProductDetailScreen.kt` - Product details screen
* `CartScreen.kt` - Cart management screen
* `SuccessScreen.kt` - Order success screen

## Requirements

* IntelliJ IDEA or Android Studio
* JDK 21
* Android SDK 35
* Minimum SDK 24

## How to Run

1. Clone the repository
2. Open the project in IntelliJ IDEA
3. Sync Gradle files
4. Select an emulator or Android device
5. Run the application

## Objective

The purpose of this project is to learn Android application development using Kotlin and Jetpack Compose while implementing navigation, state management, and responsive UI components.
